# My English Teacher Bot 🇬🇧

Телеграм-бот для разговорной практики английского языка с AI.
Помогает преодолеть страх говорить, объясняет грамматику и расширяет словарный запас в удобном стиле общения.

## Возможности

- 🗣 Разговорная практика без стеснения
- 📚 Персональный словарь
- 📖 Объяснение грамматики в 3 стилях (на пальцах / по-дружески / классический)
- 🔔 Напоминания о занятиях по местному времени
- 📊 Отслеживание прогресса и серий занятий
- ⭐ Подписка для безлимитного доступа

## Технологии

- Python 3.10+
- python-telegram-bot
- OpenRouter (AI)
- SQLite (база данных)
- APScheduler (фоновые задачи)

## Запуск (для разработки)

1. Склонировать репозиторий
2. Создать виртуальное окружение и установить зависимости: pip install -r requirements.txt
3. Скопировать `.env.example` в `.env` и заполнить своими ключами
4. Запустить бота: python -m bot.main
